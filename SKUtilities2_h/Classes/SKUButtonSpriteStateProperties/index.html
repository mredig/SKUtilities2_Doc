SKUButtonSpriteStateProperties

Conforms to:
NSCopying
Superclass:
NSObject
Declared In:
SKUtilities2

Introduction

Stores a single state for sprites in buttons. The properties set on this object gets passed to the button's sprite for the appropriate state.
